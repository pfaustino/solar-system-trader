# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| latest `main` | yes |

## Reporting a vulnerability

Do not open public issues for security problems. Use GitHub **Private vulnerability reporting** (Settings → Security) or email the maintainer.

We aim to acknowledge within 72 hours.

## Scope

This repository and its CI workflows. Third-party hosting/CDN is out of scope.
